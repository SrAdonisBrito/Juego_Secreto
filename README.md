# 🎯 Secret Number Game

## 📌 Descripción
Aplicación web interactiva donde el usuario debe adivinar un número generado aleatoriamente por el sistema.

## 🚀 Tecnologías
- HTML
- CSS
- JavaScript

## ⚙️ Funcionalidades
- Generación de número aleatorio
- Validación de intentos
- Feedback al usuario (mayor / menor)
- Reinicio del juego
